Рыболовные узлы

Освойте основные рыболовные узлы

Научитесь вязать самые важные рыболовные узлы с пошаговыми инструкциями и видеоуроками.

  • Улучшенный клинч

    Самый популярный узел для привязывания крючка, блесны или вертлюжка к монофильной или флюорокарбоновой леске. Надёжный и простой в вязке.

    Сложность: Легко
    Прочность: Высокая
    Назначение: Крючок
    Научиться вязать

  • Паломар

    Один из самых прочных и надёжных рыболовных узлов. Работает с монофилом, флюорокарбоном и плетёнкой.

    Сложность: Легко
    Прочность: Очень высокая
    Назначение: Крючок
    Научиться вязать

  • Юни

    Универсальный узел для крючков, вертлюжков и соединения двух лесок. Также известен как петля Дункана.

    Сложность: Легко
    Прочность: Высокая
    Назначение: Крючок
    Научиться вязать

  • Кровавый узел

    Классический узел для соединения двух лесок одинакового диаметра. Популярен среди нахлыстовиков для создания конических подлесков.

    Сложность: Средне
    Прочность: Высокая
    Назначение: Леска-к-леске
    Научиться вязать

  • Хирургический узел

    Простой и быстрый узел для соединения двух лесок, даже разного диаметра. Проще кровавого узла.

    Сложность: Легко
    Прочность: Высокая
    Назначение: Леска-к-леске
    Научиться вязать

  • FG-узел

    Самое прочное соединение плетёнки с поводком. Создаёт тонкий, гладкий профиль, который отлично проходит через кольца.

    Сложность: Сложно
    Прочность: Очень высокая
    Назначение: Плетёнка-к-поводку
    Научиться вязать

  • Узел Альберто

    Модифицированный узел Юни для соединения плетёнки с флюорокарбоновым или монофильным поводком. Проще, чем FG-узел.

    Сложность: Средне
    Прочность: Высокая
    Назначение: Плетёнка-к-поводку
    Научиться вязать

  • Двойной Юни

    Два узла Юни, связанных навстречу друг другу, для соединения двух лесок. Подходит как для одинаковых, так и для разных типов лески.

    Сложность: Легко
    Прочность: Высокая
    Назначение: Леска-к-леске
    Научиться вязать

  • Незатягивающаяся петля

    Создаёт фиксированную петлю на конце лески, позволяя приманке свободно двигаться для более естественной игры.

    Сложность: Средне
    Прочность: Высокая
    Назначение: Петля
    Научиться вязать

  • Отводная петля

    Создаёт петлю в середине лески для крепления крючка или отвода. Незаменима для оснасток с несколькими крючками.

    Сложность: Средне
    Прочность: Высокая
    Назначение: Петля
    Научиться вязать

  • Бимини твист

    Создаёт сдвоенный участок лески, сохраняющий почти 100% прочности. Незаменим для морской трофейной рыбалки.

    Сложность: Сложно
    Прочность: Очень высокая
    Назначение: Петля
    Научиться вязать

  • Снелл

    Привязывает леску непосредственно к цевью крючка для прямолинейного соединения. Максимальная эффективность подсечки.

    Сложность: Средне
    Прочность: Очень высокая
    Назначение: Крючок
    Научиться вязать

  • Сан-Диего джем

    Прочный, надёжный узел, любимый рыбаками Сан-Диего при ловле тунца на дальних рейсах. Отлично работает с флюорокарбоном.

    Сложность: Средне

    Назначение: Крючок
    Научиться вязать

    Научиться вязать

  • Рапала

    Сложность: Средне
    Прочность: Высокая
